To run the program

1. Run menu by typing ./menu.sh. This presents a menu.
2. Select readin to read in values from users.txt
3. Select newuser to enter one user at a time
4. Select choice to modify users
5. Select remove to delete users

The users.txt file can be modified to readin new users
the badwords function, in newuser.sh,  can be modified to add illegal user names
